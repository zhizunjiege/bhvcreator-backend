# bhvcreator-backend

## Test

Use below command to run tests:

For Linux:

```bash
./tools/unittests.sh
```

For Windows:

```bat
./tools/unittests.bat
```

## Build

Use below command to build docker image:

For Linux:

```bash
./tools/build-docker.sh
```

For Windows:

```bat
./tools/build-docker.bat
```
